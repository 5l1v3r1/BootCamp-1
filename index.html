Okay, some more details...

  • Boot camp materials can be found at the GitHub repository. This includes all lessons from this year, and lessons from past years that aren't being given this years
  • Yes, you're invited no matter whether you're an incoming student, postdoc, faculty, or if you have been in the physics department for a while!
  • The material covered on each day is organized in terms of experience level. Each week we will send out an email with a list of skills. If you feel like you already possess those skills, no need to come to that week's lessons.
  • Day 0 will be dedicated to installing and setting up unix and python environments. We will send out a list of dependencies that you will need to have installed. If you have any doubt that you have everything you need set up, then we highly encourage you to come. We will be spending little to no time troubleshooting during the Wednesday sessions.
  • No, no background in Unix or Python is required. You are very encouraged to attend the sessions on the first two Wednesdays if you are not familiar with Unix or Python.
  • The last day will have a larger focus on KIPAC related topics, but should still be interesting to a wide variety of physicists.
  • We will try to make these sessions as hands-on as possible, so bring your laptop to the sessions.
  • You can contact the organizer, Sean McLaughlin, at swmclau2 [at] stanford [dot] edu.
